ECLIPSE项目导入ANDROID STUDIO错误总结和STUDIO使用心得

Eclipse项目导入Android Stuio配置出现

Error:Timeout waiting to lock buildscript class cache for build file 'H:\studioproject\Generic_SN\build.gradle'(C:\Users\Administrator\.gradle\caches\2.2.1\scripts\build_81ep6udn3nlzszbotl32uedjz\ProjectScript\buildscript). It is currently in use by another Gradle instance.

Owner PID: unknown
Our PID: 8496
Owner Operation: unknown
Our operation: Initialize cache
Lock file: C:\Users\Administrator\.gradle\caches\2.2.1\scripts\build_81ep6udn3nlzszbotl32uedjz\ProjectScript\buildscript\cache.properties.lock

找到红色标识部分文件然后删除文件。删除的时候会提示文件被占用。开打进程管理器把关于java的进程结束后，在Android

Stuido中重新build就可以。
