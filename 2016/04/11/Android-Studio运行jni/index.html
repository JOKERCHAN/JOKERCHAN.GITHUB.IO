Android Studio运行jni

发表于 2016-04-11   |   分类于 Android   |     |   阅读次数

工作有一段时间了,但是工作中从来也没接触到过jni的开发。今天忽然看见群里有人再说就去看了看,这里使用android stuido的环境来做一下记录。

首先新建一个名为JNISTUDY的项目。

然后建立一个JniUtils类,声明native方法

package com.jokerchan.jnistudy;

/**
 * Created by JOKERCHAN on 2016/4/11.
 */
public class JniUtils {

    static {
        System.loadLibrary("jokerlib");
    }

    public native String getStringFromNative();

}

public native String getStringFromNative();就是申明的native方法，上面一段代码过会再解释。

然后在按如图rebuild project

然后我们会在看到这个目录JNISTUDY\app\build\intermediates\classes\debug。
然后接下来的步骤就是根据生成的class文件，利用javah生成对应的.h头文件。
我们使用studio的Terminal控制台进入JNISTUDY\app\build\intermediates\classes\debug目录
直接 cd app\build\intermediates\classes\debug。
然后执行javah -jni com.jokerchan.jnistudy.JniUtils

执行完命令后可以通过上图看见已经在刚才的目录中生成了.h的头文件。

内容是

/* DO NOT EDIT THIS FILE - it is machine generated */
#include <jni.h>
/* Header for class com_jokerchan_jnistudy_JniUtils */

#ifndef _Included_com_jokerchan_jnistudy_JniUtils
#define _Included_com_jokerchan_jnistudy_JniUtils
#ifdef __cplusplus
extern "C" {
#endif
/*
 * Class:     com_jokerchan_jnistudy_JniUtils
 * Method:    getStringFromNative
 * Signature: ()Ljava/lang/String;
 */
JNIEXPORT jstring JNICALL Java_com_jokerchan_jnistudy_JniUtils_getStringFromNative
  (JNIEnv *, jobject);

#ifdef __cplusplus
}
#endif
#endif

现在我们在app->src->main->java下建立一个jni目录然后把刚才的头文件拷贝过去。然后新建一个名字为
jnitest.c的文件，然后我们去实现本地的方法。方法如下

#include <com_jokerchan_jnistudy_JniUtils.h>

JNIEXPORT jstring JNICALL Java_com_jokerchan_jnistudy_JniUtils_getStringFromNative
  (JNIEnv *env, jobject obj){
     return (*env)->NewStringUTF(env,"This just a test for Android Studio NDK JNI developer of JokerChan!!!");
  }

这里我们返回了一个String内容为This just a test for Android Studio NDK JNI developer of JokerChan!!!。
现在我们去build.gradle文件中去配置ndk相关内容。配置如图:

然后我们需要在gradle.properties文件末尾添加一句
android.useDeprecatedNdk=true。

前面说到的

static {
        System.loadLibrary("jokerlib");
    }

这句话里面的jokerlib为buidle.grald里面的moduleName即jokerlib。

现在我们运行项目可以看见已经生成了.so文件。

我们用TextView把内容显示出来

package com.jokerchan.jnistudy;

import android.app.Activity;
import android.os.Bundle;
import android.widget.TextView;


/**
 * android studio ndk测试
 * @version 1.0
 * @author jokerchan
 * @since 2016年4月11日 11:22:05
 */
public class MainActivity extends Activity {

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
        TextView tv = (TextView)findViewById(R.id.show);
        JniUtils ju = new JniUtils();
        tv.setText(ju.getStringFromNative());
    }
}

基本流程已经跑通了。
