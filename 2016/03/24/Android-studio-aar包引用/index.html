Android studio aar包引用

发表于 2016-03-24   |   分类于 Android   |     |   阅读次数

以前打包总是打成jar格式的包,但是有时候想要资源文件一起打包的时候就显得不足了。

jar和arr的不同

jar包只能打入class文件。现在就没有那么麻烦了，Android studio给我们提供一种更简便的方式，可以同时把class、res、lib文件打包进去，现在的support v7 和gson-2.21就是aar打包方式。

生成aar

新建一个model为library然后选择rebuild，然后会在build->outputs->aar目录下面找到生成的aar包。

项目中引用

1.复制aar到libs目录中

2.在app的buidle.gradle中添加

repositories {
    flatDir {
        dirs 'libs'
    }
}

3.在dependencies中添加 compile(name: 'volley-debug', ext: 'aar')

4.重写编译一次会发现已经可以使用了。如果所示：

#arr
